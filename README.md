# PoE 2 Farm Tracker 🎮📊

A lightweight, efficient, and productivity-focused farm tracker for **Path of Exile 2**, designed to run directly in the browser or as a quick-access companion tool during gameplay.

---

## ✨ Features

* **⏱️ Integrated Timer**: Start, pause, and reset your run times easily, with automatic calculation of **Divines per hour (Div/h)**.
* **📁 Preset System**: Create and manage different farming strategies (e.g., *Ritual*, *Sehkemas*, etc.) with dynamic switching.
* **🔄 "All" View**: Filter the complete history and view the unified sum of all items and profits obtained across all your presets.
* **📦 Item Control & Customization**: Add, rename, reorder (via *Drag and Drop*), and set unit prices and quantities for drop items.
* **⚡ Global Hotkeys**: Configure custom keyboard shortcuts to speed up item counting and timer controls.
* **📋 Discord Summary**: Copy a formatted report of your runs with a single click to share with friends.
* **💾 Data Backup**: Export and import all your data and histories easily via `.json` files.
* **⚙️ Adjustable Layout**: Options for resizing (S, M, L), vertical/horizontal orientation, and opacity control.

---

## 🚀 How to Use

1. Download the main project file (`.html`).
2. Open the file in any modern web browser (Chrome, Edge, Firefox, etc.).
3. Create your first preset in the **Presets** tab and start tracking your farming progress!
